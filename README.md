# 🚀 Entrepreneurship Development Cell (E-Cell) SGIP Panhala - Website

This repository contains the official website developed for the **Entrepreneurship Development Cell (E-Cell), Sgip panhala**.  
Our mission is to foster innovation, support startups, and promote entrepreneurial culture among students and beyond.

---

## 🌟 Features
- 🎯 Homepage highlighting E-Cell’s mission and vision  
- 📅 Events section with details and registration  
- 📰 Blogs & resources for aspiring entrepreneurs  
- 🤝 Startup support programs and opportunities  
- 📌 Contact & networking information  
- 📱 Fully responsive design for all devices  

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS, JavaScript  
- **Backend:** Node.js   
- **Database:**  MySQL
